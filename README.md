## Project Overview
The Online Voting System is a Python-based application that manages a simple
electronic voting process. It ensures one voter can vote only once and provides
accurate vote counting.

## SDLC Implementation

### Requirement Analysis
The system registers voters and candidates, allows voting, and displays results.

### System Design
Object-oriented design using:
- Voter
- Candidate
- VotingSystem

### Implementation
Language: Python  
File: online_voting_system.py

### Testing
Manual testing was used to validate voting rules and vote counting.

### Deployment
Runs on any system with Python 3 installed.

### Maintenance
Can be extended with databases, security, and web interfaces.
